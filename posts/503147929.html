毕业旅行之云南

发表于2024-05-27
|字数总计:3k|阅读时长:10分钟|阅读量:

写在开始

这场旅行期待很久了，准备工作其实还是蛮多的，主要参考来自b站各个up的旅游视频

云南我觉得大理≥昆明>>>丽江，因此本来安排给泸沽湖的时间给了大理

旅行是我们漫漫人生中暂时离开既有生活的闪亮间隙，在旅行中经历另一种生活形态，与这辈子可能不会再见的人与事共同创造难忘的人生记忆

喜欢大理，在那里有睡到自然醒的松弛，有走哪算哪的随意，有抚平心灵的归属感，有些记忆最后封存在洱海就好

高德地图链接

昆明

day1(5.19)

✈️MU5853 成都天府T2→昆明长水 10:00

7:00左右出发 打车到火车南站换乘18号线

我们出门太慢了，而且行李箱大，路上耽误了很久，差两分钟就值不上机了，最好提前一点走

📍行程：机场→住宿点1→董家湾→西南联大旧址(需要提前预约)→云南陆军讲武堂→翠湖公园→黄公东街→南屏步行街→住宿点1

🏨住宿点1 春天印象写字楼内

🍲午饭：三胖边境小食堂

这两没吃上：

住宿点1附近 🍲刀敬小锅米线(永善路店)

云南陆军讲武堂附近 🍲茴香云南盛·过桥米线

云南陆军讲武堂：周二到周日开放，下午四点半禁止入场，军人提枪表演时间：9点、10点、11点，下午14点、15点、16点

我们在西南联大旧址呆太久了，没来得及进场，西南联大那边有章可以盖

翠湖公园：最近应该没有海鸥了，但是还是有松鼠，西门进，南门出

🚩黄公东街一个比较火的拍照点，导航到朝茶暮酒

对月楼：已经变成一个酒店了，旁边和人民英雄纪念碑紧挨，也叫纸片楼

🍲晚饭：西双曼傣·傣味野生菌手抓饭

去南屏街和大妈们一块跳了会舞就回了

账务

✈️机票 成都天府→昆明长水 282.75

🏨住宿 287.28/4 = 71.82

🚕打车 51.92/4 = 12.98

🍲吃饭 65+228+12(西瓜) = 305/4 = 76.25

合计 282.75+71.82+12.98+76.25 = 443.8

大理

day1(5.20)

🚅昆明→大理，动车D8688，住的地方离昆明站就几分钟，下午1:05到达大理站

📍行程：住宿点1→昆明站→大理站→住宿点2→大理古城(洱海门→人民路→大理天主教堂→五华楼)→住宿点2

🏨住宿点2 才村小邑庄，近才村码头

下了高铁直接打车去住宿点，四个人基本全程打车了

午饭直接在民宿楼下吃的，老板手艺很好，还给我们送菜了

🍲午饭：漫时光艺术花园餐厅，我们住宿就在它楼上

老板做饭真的绝杀，太好吃了

运气非常好，到大理的时候还在哗啦哗啦下大雨，等到四点多我们出门已经基本没雨了，这时候的大理古城有一种别样的美

ℹ️大理天主教堂：建筑风格相当混血，具有本地特色

ℹ️大元帅府：每天晚上8:00-9:00门口有篝火晚会，没有篝火就是嬢嬢们跳广场舞了

ℹ️五华楼：到城墙上可以稍微俯看古城

ℹ️广武路：比较多小饰品店

ℹ️人民路：算是大理流浪文化的发源地吧，晚上九点以后比较热闹

ℹ️床单艺术中心：买的都是手作小物品，但是只有周末才有(近南门)

🍲晚饭：德胜桥（大理古城店）

味道还可以，性价比适中

吃完回民宿已经快12点了，大理古城还是相对商业化的，往北边的北门菜市场走走也挺好

账务

🚅昆明→大理 145

🏨住宿 5天合计 954.89+540 = 1494.89/4 = 373.72

🚕打车 5.49+38.97+10.34+19.4 = 74.2/4 = 18.55

🍲吃饭 (44+88)/4 + 36.9/2 = 51.45

合计 145+373.7225+18.55+51.45 = 588.72

day2(5.21)

时间这样就很充沛了，于是今天只沿海西线逛了逛，也没去喜洲古镇

📍行程：住宿点2→日出→生态廊道骑行(住宿点2→才村码头→观海庭→下波棚→潘溪村S弯→廊桥)

🌅日出地点导航大理屿月民宿，那里人少，感觉日出效果挺好，但是要注意别踩水里了(别问我为什么这么说π_π)

在大理看了三天的日出，真的日出不看白来，每次都有巨大的心灵震撼

🍲早饭是在民宿吃的，老板整的包子和小米粥，因为我们住了三天以上早餐是免费的

在洱海生态廊道可以找电动车，双人的60/h，可以连蓝牙放音乐，注意找电够的

自行车是预付30，前15分钟2r，后面15分钟1.5r

直接跳过午饭，晚饭在民宿楼下吃的，吃完在生态廊道散步，这种感觉很棒

账务

🛵洱海电动车 60

🍲吃饭 48/2 = 24

合计 60+24 = 88

day3(5.22)

📍行程：住宿点2→喜洲古镇(喜洲菜市场→严家大院→蓝州扎染体验店→转角楼→禾田里(大树的位置))→住宿点2

在喜洲菜市场体验了一下生猪肉，感觉需要一点勇气

在严家大院刚好碰见旅游团，直接听了他们的讲解，感觉12r的门票很值

扎染需要预留2h及以上时间，那天和甜呀一起缝绘裙子差不多花了四个小时才结束，感觉效果还是很好的

这个时候天已经都黑了，秉持着来都来了的想法，摸黑去看了下麦田和刘亦菲同款大树，晚上去看这树倒也挺奇特

账务

🚕打车 1+50.2+44.14 = 95.34/4 = 23.84

🎫严家大院 12

🥻扎染 90

🍲吃饭 97/4 = 24.25 +3+10 = 37.25

合计 23.835+12+37.25+90 = 163.09

day4(5.23)

📍行程：住宿点2→三月街→苍洱大道→岛七白族菜→住宿点2→洱海散步

刚好碰上农历16，三月街有赶集活动，于是去体验了一波，感觉物价还是可以的，没有很贵，很实惠

ℹ️去苍洱大道导航财校路口(公交站)，不然要爬坡

🍲岛七白族菜 这家店还是很不错的，位置有点偏，特色菜都好吃，性价比高

领悟了大理旅行的真谛就是沿着洱海散步发呆，坐在海边可以呆一天，这也就是为什么day5在这没有再写的原因了

在大理的5天看了三场日出，每场都让我很震撼，来大理一定要看一场日出！！！

账务

🍢零食 (10.5+10)/2 = 10.25

🚕打车 16.06+6.49+8.04+7.29 = 37.88/4 = 9.47

🍲吃饭 15+170 = 185/4 = 46.25

合计 10.25+9.47+46.25 = 65.97

day5(5.24)

不太想去苍山，爬山有点小累

没想着再到处跑了，就在洱海边再骑了骑车，很喜欢洱海给我的感觉

账务

🍲吃饭 68/2+250/5 = 84

🚲️共享自行车 10

海东线(未走)

📍行程：住宿点2→双廊古镇→(鹿卧山→)挖色镇→小普陀→文笔村→圣托里尼理想邦(到时候可以考虑不去)→兴盛大桥→下关→住宿点2

海东线可以包车，还还价300左右还可以接受，小红书上次问了一下包车时间是在9:00~18:00

打车路线：大理古镇到双廊古镇110左右，双廊古镇到鹿卧山那块25左右，鹿卧山到小普陀那块10左右；小普陀到文笔村22左右（日落）；小普陀到圣托里尼理想邦40左右；圣托里尼到下关48左右；下关到大理古镇30左右

合计也在300左右，但更自由一些，而且可以到时候叠加各种打车券

午饭在双廊古镇解决

ℹ️鹿卧山：废弃窗户，洱海之窗，山洞悬崖，风评不一

ℹ️小普陀上岛要花钱，没啥必要

ℹ️文笔村可以看日落

东线没啥太多特色

下关🍲云龙黑山羊野生菌火锅(嘉士伯大道店) 不点套餐，他家羊肉不错，搞一点炒菜

巍山线(未走)

住宿点2→洱海月湿地公园→下关吃午饭→大理站→巍山古城

下关那边到大理站就十来分钟打车，高铁到巍山站约22分钟，可以考虑体验一下一等座34r，二等座22r

大理过去的最好可以坐，巍山回的最后一班高铁是18:47的

丽江

由于去年刚去的川西，对雪山有些审美疲劳，因此没把玉龙雪山列入我们的行程

day1(5.25)

📍行程：住宿点2→三月街口(公交站)→丽江古城南门→住宿点3→白沙古镇→清溪公园→九婆餐饮店→住宿点3

🏨住宿点3在丽江古城西边，因为我们不咋去丽江古城在这边还便宜

🍲午饭 纳西纯杂锅菜(白沙古镇店) 便宜好吃，点了一个中锅+一份炸凉粉

白沙古镇可以走完，差不多两个小时，白沙壁画我们没来得及去

不想去丽江古城于是去了一个居民公园——清溪公园，但是这里的日落和湖却特别的好看

🍲晚饭 九婆餐饮店 腊排骨，味道还不错

账务

🏨住宿 407.8/4 = 101.95

🚌大巴 29.5

🚕打车 14.58+5.54+26.1+7.8+14.8 = 68.82/4 = 17.21

🍲吃饭 98+98+10 = 206/4 =51.5

合计 101.95+29.5+17.205+51.5 = 200.16

day2(5.26)

出了一些意外今天没出门

🍲晚饭 老建水胖子烧烤 可以避雷了，不是很好吃，上菜慢的想死

账务

✈️机票 丽江三义→成都双流 460

🍲吃饭 252/4 = 63

合计 460+63 = 523

泸沽湖线(未走)

📍行程：住宿点2→泸沽湖住宿点3(约4h)→大洛水村→云南情人滩

🏨住宿点3考虑在大洛水村，可以离湖近一点，而且车可以给送到住宿的地方

泸沽湖景区门票35r(学生票)，包车一天200r，可以找民宿老板或者当天送我们过去的司机问下

泸沽湖最有名的就是石锅蒸汽鱼 🍲菌香小院·蒸汽石锅鱼

📍行程：住宿点3→绕湖半日游(5h左右)→白沙古镇(约4h)住宿点4

顺时针：大洛水→云南情人滩→里格半岛→女神湾→走婚桥→草海

逆时针：大洛水→草海→走婚桥→女神湾→里格半岛→云南情人滩

女神山到时候看情况再看看要不要上索道上山顶看

ℹ️里格半岛：可以看半全景

ℹ️女神山：上索道可以看到全景，缆车比较刺激()，不是那种封闭小车

ℹ️走婚桥：是不能走回头路的，不然会找不到对象😿

回可以坐进来时候一家公司的车，大概80/人

注意事项

⚠️大理算半个高原，需要及时做好防晒，可以带墨镜+帽子，防晒霜和芦荟胶最好都带上

📖可以带着盖章册，旅游城市的邮局肯定有章，不细找了，碰到就盖

👚云南嘛，四季如春，要稍微带些长袖和外套，昼夜温差巨大

最终账单合计
