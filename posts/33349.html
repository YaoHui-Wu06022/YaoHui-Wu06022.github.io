git拉取上传代码

发表于2025-09-01
|字数总计:128|阅读时长:1分钟|阅读量:

  1. 打开项目，此时项目中是没有 .git 文件的

  2. 在【鼠标右击】弹出的菜单中，点击【Git Bash Here】

  3. 在命令窗口中输入：git init

  4. 在命令窗口中输入：git remote add origin (仓库地址) [相当于绑定仓库了]

    输入：git branch -M main(仅第一次)

  5. 添加需要提交的文件：git add (目录下文件) or git add.

  6. 准备上传：git commit -m “备注信息”

  7. 提交上传请求：git push origin main

以后上传只需要 5-7 步即可
