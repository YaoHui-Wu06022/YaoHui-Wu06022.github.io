秦皇岛

那么近那么美周末到河北(doge)

高德地图链接

🏖️day1：

T5603北京站 -> 北戴河站(10:48-13:25) -> 租电动车 -> 蔚蓝海岸 -> 🌇老虎石看日落 -> 绕海边骑行 -> 海鲜自助(遇见) -> 酒店

蔚蓝海岸：免费浴场，但是脏的很，玩完沙滩冲脚2r/min，一般，有个猫的天空之城图书店还可以，可以盖章

老虎石：好看，人多，旺季根本没机会停电动车，建议淡季时候再考虑去这看日落，不过确实好看

海鲜自助：遇见有学生价，而且一般海鲜自助都是吃两个小时，这家店整体还可以，也有烤榴莲，吃了四大锅包回本的（皮皮虾一般，生蚝太好了）

北戴河电动车管的特别严，需要严格戴头盔and规定停车区停车，不然要被拖车且罚钱

租电动车特别划算！！！

🧾账单：

  • 🚅T5603：41.5

  • 🛵电动车：41.9/2

  • 🛌酒店：101/2

  • 🦞海鲜自助：132 (划算)

合计 244.95

🌊day2

🌅海天一色(起不来没看成) -> 酒店 -> 北戴河站 -> 还车 -> G1299山海关站(10:20-10:47) -> 邱记太子锅 -> 老龙头景区 -> 打车回金梦海湾海滩旁酒店 -> 秦皇小巷

老龙头：付费的还是比免费的海岸好得多，好玩好看且干净，值得一去

海鲜饺子：一般，但是吃的确实饱，可能是因为前一天海鲜吃太多了

秦皇小巷：干净的小吃街，价格没有太离谱，但是也没有特别好吃

市区这边电动车太贵了，如果第一天就到这边的可以考虑租，街上的一小时19，建议按天租，或者直接共享助力车3r/30min

🧾账单：

  • 🚅G1299：20
  • 🚕打车：29.46
  • 🛌海景公寓：147.84/2 淡季超划算，阳台可以看海
  • 🥘太子锅：179/2
  • 🎫老龙头门票：25 (学生+美团优惠)
  • 🥟海鲜饺子：94/2

合计 284.88

day3

西港花园(熬夜了没去成) -> 秦皇岛站返北京

🎒带的：

  • 🕶️墨镜

  • 🩴洞洞鞋

  • 🧴防晒霜
